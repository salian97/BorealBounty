# BorealBounty

The player plays as Fifendekel, a hungry squirrel looking for acorns before winter strikes! Explore the forest of Freya and encounter lovable characters that test your wisdom. Solve riddles and answer questions that adapt to your skills in real time. earn acorns from villagers, and survive the winter to become the proudest chubby squirrel in the forest.

This is a video game built in Unity, meant to be played on Windows, Mac, or Linux.

As of this current revision, there is no official build for any of the aforementioned platforms. You must clone this repository and use Unity version 2022.3.6f3 to build from source.

We tested this project with the NeuroPawn headset, and is currently our only supported board. If you would like to add more boards or test with a synthetic board, you may edit the script called Assets/Scripts/EEGDataAnalyzer.cs or use the inspector.
